Quick Start দ্রুত শুরু

  1. Install AdorLipi from source or package.
  2. Settings → Keyboard → Input Sources → Add Bangla → AdorLipi.
  3. Switch input source with Super + Space.

Tip: Use python3 cli/main.py for quick local engine testing.
